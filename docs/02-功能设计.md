# Polymarket Go 高层功能设计文档

**文档版本**: v1.0
**更新日期**: 2025-11-25
**设计目标**: 设计面向策略开发者的高层 API，简化 Polymarket 交易策略开发

---

## 1. 设计理念

### 1.1 核心原则

**易用性优先**: 隐藏底层复杂性，提供直观的 API
**数据驱动**: 提供丰富的数据分析和统计功能
**策略友好**: 专门为量化策略开发优化
**安全可靠**: 内置风险控制和资金管理机制
**高性能**: 支持实时数据和批量操作

### 1.2 用户画像

**主要用户**: 量化策略开发者
**核心需求**:
- 快速获取账户资产概览
- 统计交易绩效（胜率、盈亏比、收益率）
- 监控持仓和风险
- 发现交易机会
- 执行交易策略
- 管理资金分配

## 2. 架构设计

### 2.1 模块划分

```
polymarket.Client
├── Portfolio()      // 持仓管理模块
```

### 2.2 模块职责

| 模块 | 职责 | 底层依赖 |

### 3.4 实现策略

**数据聚合流程**:
1. 调用 Data Client 获取持仓列表
2. 并行调用 Gamma Client 获取市场元数据
3. 并行调用 CLOB Client 获取实时价格
4. 计算衍生字段（市值、盈亏、评分）
5. 返回富化后的持仓数据

**缓存策略**:
- 市场元数据: 缓存 5 分钟
- 价格数据: 缓存 10 秒
- 持仓数据: 按需刷新

```go
    MaxDrawdown     decimal.Decimal
```

### 4.4 计算方法

**胜率计算**:
```
WinRate = WinTrades / TotalTrades * 100%
```

**盈亏比计算**:
```
ProfitFactor = TotalProfit / TotalLoss
AvgWinLossRatio = AvgProfit / AvgLoss
```

**夏普比率**:
```
SharpeRatio = (AvgReturn - RiskFreeRate) / StdDev
```

**最大回撤**:
```
MaxDrawdown = (Trough - Peak) / Peak * 100%
```

### 5.4 推荐算法设计

**市场评分算法**:
```go
func calculateMarketScore(market *EnrichedMarket) float64 {
    liquidityWeight := 0.3
    volumeWeight := 0.3
    spreadWeight := 0.2
    activityWeight := 0.2

    score := market.LiquidityScore * liquidityWeight +
             market.VolumeScore * volumeWeight +
             (100 - market.SpreadScore) * spreadWeight +  // 价差越小越好
             market.ActivityScore * activityWeight

    return score
}
```

**流动性评分**:
```go
func calculateLiquidityScore(orderBook *OrderBookSummary) float64 {
    // 订单簿深度评分
    bidDepth := sumOrderBookDepth(orderBook.Bids, 5)  // 前5档
    askDepth := sumOrderBookDepth(orderBook.Asks, 5)
    totalDepth := bidDepth + askDepth

    // 归一化到 0-100
    return math.Min(totalDepth.Float64() / 10000 * 100, 100)
}
```

### 6.4 智能订单实现

**TWAP 订单**:
```go
func (t *Trading) executeTWAP(ctx context.Context, req SmartOrderRequest) (*SmartOrderResult, error) {
    sizePerInterval := req.TotalSize / decimal.NewFromInt(int64(req.Intervals))
    intervalDuration := req.Duration / time.Duration(req.Intervals)

    results := make([]*OrderResult, 0, req.Intervals)

    for i := 0; i < req.Intervals; i++ {
        // 下单
        order := MarketOrderRequest{
            TokenID: req.TokenID,
            Side:    req.Side,
            Amount:  sizePerInterval,
        }

        result, err := t.PlaceMarketOrder(ctx, order)
        if err != nil {
            return nil, err
        }

        results = append(results, result)

        // 等待下一个间隔
        if i < req.Intervals-1 {
            time.Sleep(intervalDuration)
        }
    }

    return aggregateResults(results), nil
}
```

**Iceberg 订单**:
```go
func (t *Trading) executeIceberg(ctx context.Context, req SmartOrderRequest) (*SmartOrderResult, error) {
    remaining := req.TotalSize
    results := make([]*OrderResult, 0)

    for remaining.GreaterThan(decimal.Zero) {
        displaySize := decimal.Min(req.DisplaySize, remaining)

        order := LimitOrderRequest{
            TokenID:   req.TokenID,
            Side:      req.Side,
            Price:     req.PriceLimit,
            Size:      displaySize,
            OrderType: "GTC",
        }

        result, err := t.PlaceLimitOrder(ctx, order)
        if err != nil {
            return nil, err
        }

        results = append(results, result)
        remaining = remaining.Sub(result.FilledSize)

        // 等待成交
        if err := t.waitForFill(ctx, result.OrderID); err != nil {
            return nil, err
        }
    }

    return aggregateResults(results), nil
}
```

### 7.4 风险计算方法

**仓位风险评分**:
```go
func calculatePositionRisk(size, portfolioValue, maxPosition decimal.Decimal) float64 {
    positionRatio := size.Div(portfolioValue)
    maxRatio := maxPosition.Div(portfolioValue)

    if positionRatio.GreaterThan(maxRatio) {
        return 100.0  // 超过限制
    }

    return positionRatio.Div(maxRatio).Mul(decimal.NewFromInt(100)).Float64()
}
```

**Kelly 仓位计算**:
```go
func calculateKellyPosition(winRate, avgWin, avgLoss, capital decimal.Decimal) decimal.Decimal {
    // Kelly Formula: f = (p * b - q) / b
    // p = 胜率, q = 1-p, b = avgWin/avgLoss
    p := winRate
    q := decimal.NewFromInt(1).Sub(winRate)
    b := avgWin.Div(avgLoss)

    kellyPercent := p.Mul(b).Sub(q).Div(b)

    // 使用半凯利或1/4凯利以降低风险
    kellyPercent = kellyPercent.Div(decimal.NewFromInt(2))

    return capital.Mul(kellyPercent)
}
```
