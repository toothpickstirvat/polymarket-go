## 1. 项目概览

### 1.1 项目目标

构建一个**面向策略开发者**的 Polymarket Go 高层封装库，核心目标：

✅ **简化复杂性**: 隐藏底层 5 个客户端的复杂交互
✅ **提升效率**: 提供开箱即用的数据分析和统计功能
✅ **降低门槛**: 让策略开发者专注于策略逻辑，而非底层细节
✅ **保证质量**: 高测试覆盖率，稳定可靠

### 1.2 核心价值

| 用户需求 | 当前痛点 | 解决方案 |
|---------|---------|---------|
| 账户资产概览 | 需要调用多个 API 手动聚合 | Portfolio.GetSummary() 一键获取 |
| 交易绩效统计 | 需要自己计算胜率、盈亏比 | Analytics.GetPerformanceOverview() |
| 市场发现 | 需要手动筛选和评估市场 | MarketFinder.GetRecommendedMarkets() |
| 交易执行 | 复杂的订单创建和签名流程 | Trading.PlaceMarketOrder() 一步完成 |
| 风险控制 | 缺少自动化风险评估 | RiskManager.EvaluateOrderRisk() |
| 实时监控 | 需要自己管理 WebSocket 连接 | Monitor.SubscribePriceUpdates() |
